# Tools

This folder is for developer scripts, e.g.:
- Generate TypeScript client types from the FastAPI OpenAPI spec
- Run format/lint across the monorepo
- `setup-and-run.sh`: check local requirements, install deps, and run API + web together
  - now detects Linux/macOS and can prompt to install missing prerequisites
- `context-refresh.sh`: quick context handoff checklist for AGENTS.md + docs sync
