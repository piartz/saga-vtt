# Vision

Build a **browser-based, server-authoritative virtual tabletop** focused on miniature skirmish play:
- Continuous measurement (not grid-locked)
- Token bases with real diameters
- Clear audit trail of dice and actions
- Smooth UX for “select → measure → move → confirm”
- Extensible rules layer (pluggable modules)

Non-goals for early MVP:
- 3D rendering
- photorealistic table textures
- full army list builder
- integrated payment / licensing systems

The first goal is to enable **two players** to complete a full game flow online with:
- a shared board state
- deterministic rules enforcement
- readable action log
